<details>
<summary><strong>What you can learn</strong></summary>
<span>
How assets flow through the API — from upload to alignment to synced playback.
</span>
</details>
<details>
<summary><strong>What is an asset</strong></summary>
<span>
The source [**audio or video**](https://developer.audioshake.ai/supported-format) file publicly hosted or provided via an AWS pre-signed URL.
</span>
</details>
<details>
<summary><strong>How to use this demo</strong></summary>
<span>
* Create or select an asset  
* Choose an alignment or create a new one  
* Play media with synced lyrics/transcription  
* Optionally edit words and download the updated JSON
</span>
</details>
<details>
<summary><strong>Demo Asset Options</strong></summary>
<span>
* Use built-in **Demo Assets**  
* Paste a public media URL  
* Drop a `demo-assets.json` data file created with our [**create-demo-assets**](https://www.npmjs.com/package/create-demo-assets) tool.
</span>
</details>
<details>
<summary><strong>How to create an alignment</strong></summary>
<span>
* Select an asset  
* Create an alignment or select alignment from the list. 
* Edit if needed and download the JSON
</span>
</details>
<details>
<summary><strong>API Console & Code Examples</strong></summary>
<span>

**View Code** shows JS, Node, Curl, Python, and Swift examples with your API key auto-injected.

**Console** provides:  
1. **List** — list all alignments  
2. **Get Task** — fetch by Task ID  
3. **Create Task** — Uses currently selected asset to create a new task.
</span>
</details>
