### **AudioShake API Alignment Demo**

This demo shows how the AudioShake API converts a supported [audio or video](https://developer.audioshake.ai/supported-format) file (an **asset**) into a time-aligned JSON transcript/lyric map for synced playback and editing.</br> To get started, simply **authorize** this app with your API key, load an asset into our media player and create your first Alignment.
